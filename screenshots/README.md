# Screenshots Folder

Folder ini berisi screenshot aplikasi untuk dokumentasi.

## Screenshot yang Diperlukan:

1. **main-interface.png**

   - Screenshot tampilan utama aplikasi
   - Menampilkan ketiga card: Detection, Diagnosis, dan Chat
   - Resolusi: 1920x1080 atau lebih tinggi

2. **detection-process.png**

   - Screenshot proses upload dan deteksi gambar
   - Menunjukkan flow: upload → predict → result
   - Capture pada saat menampilkan hasil prediksi

3. **diagnosis-detail.png**

   - Screenshot diagnosis card yang sudah terisi
   - Menampilkan informasi lengkap penyakit
   - Include: nama penyakit, deskripsi, penyebab, gejala, penanganan

4. **ai-chat.png**
   - Screenshot interaksi dengan AI assistant
   - Menampilkan beberapa chat messages
   - Menunjukkan pertanyaan dan jawaban yang relevan

## Cara Mengambil Screenshot:

1. Jalankan aplikasi: `npm run dev`
2. Buka browser di `http://localhost:3000`
3. Upload gambar daun padi
4. Lakukan prediksi
5. Chat dengan AI assistant
6. Ambil screenshot pada setiap tahapan

## Tools untuk Screenshot:

- **macOS**: Cmd + Shift + 4
- **Windows**: Windows + Shift + S
- **Linux**: Flameshot / GNOME Screenshot
- **Browser DevTools**: Capture full page screenshot

## Tips:

- Gunakan gambar daun padi yang jelas
- Pastikan UI terlihat dengan baik (tidak terpotong)
- Gunakan light theme untuk dokumentasi
- Crop screenshot untuk fokus pada area penting
- Compress gambar untuk mengurangi ukuran file

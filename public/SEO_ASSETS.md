# SEO Assets

This folder contains SEO-related images for better social media sharing and PWA support.

## Required Images:

### 1. Open Graph Image
- **File**: `og-image.png`
- **Size**: 1200×630 pixels
- **Purpose**: Displayed when sharing link on social media (Facebook, LinkedIn, etc.)
- **Design**: Should include:
  - RiceGuard AI logo/branding
  - Rice leaf image
  - Text: "AI-Powered Rice Disease Detection"
  - Visual elements showing the app interface

### 2. PWA Icons
- **File**: `icon-192.png`
- **Size**: 192×192 pixels
- **Purpose**: App icon for mobile devices (Android)

- **File**: `icon-512.png`
- **Size**: 512×512 pixels
- **Purpose**: App icon for mobile devices (high resolution)

### 3. Favicon (Optional but Recommended)
- **File**: `favicon.ico`
- **Size**: 32×32 pixels
- **Purpose**: Browser tab icon

## How to Create These Images:

1. **Use Figma or Canva**:
   - Create 1200×630 design for OG image
   - Export as PNG
   - Optimize with TinyPNG or ImageOptim

2. **For Icons**:
   - Create square design (1024×1024)
   - Export at different sizes
   - Ensure it looks good at small sizes

3. **Quick Tools**:
   - Canva: https://www.canva.com/
   - Figma: https://www.figma.com/
   - OG Image Generator: https://og-image.vercel.app/

## Temporary Solution:

If you don't have these images yet, you can:
1. Take a screenshot of your app
2. Crop to 1200×630 for OG image
3. Create simple icon with app logo for PWA icons
